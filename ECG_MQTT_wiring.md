## 電源軌接線

使用**右側電源軌**，左右電源軌預設不相通。

| 線色 | 從 | 到 | 說明 |
|---|---|---|---|
| 紅色 | ESP32 懸空腳 排1（3V3） | 右側電源軌 `+` | 3.3V 正極 |
| 黑色 | a欄 排1（GND） | 右側電源軌 `-` | 接地① |
| 黑色 | b欄 排7（GND） | 右側電源軌 `-` | 接地② |

## AD8232 接線

AD8232 插在 **j欄 排22~27**，從 i欄同排接線出去。

```
j欄腳位      排數    腳位名稱    接線去向
j22          22      GND        i22 → 右側電源軌 -
j23          23      3.3V       i23 → 右側電源軌 +
j24          24      OUTPUT     i24 → ESP32 懸空腳 排3（SVP/GPIO36）
j25          25      LO-        i25 → ESP32 懸空腳 排5（P34/GPIO35）
j26          26      LO+        i26 → ESP32 懸空腳 排4（SVN/GPIO34）
j27          27      SDN        不接
```

## LCD 1602A（I2C 模組）接線

LCD I2C 模組針腳順序（由上到下）：`GND VCC SDA SCL`

針腳太短，使用**母對公杜邦線**橋接到 b欄 排22~25。

> ⚠️ **VCC 必須接 5V，不可接 3.3V。** 此款藍底 1602A 為 5V 規格，接 3.3V 時字會極淡、對比無法調好（實測確認）。

```
LCD腳位    橋接到     中繼跳線              最終目的地
GND        b欄排22                         右側電源軌 -
VCC        b欄排23                         ESP32 5V 腳（懸空排19，USB端）★改 5V
SDA        b欄排24    c欄排24 → c欄排6     a欄排6（P21/GPIO21）
SCL        b欄排25    c欄排25 → c欄排3     a欄排3（P22/GPIO22）
```

> **說明：** SDA/SCL 利用麵包板橫向連通特性，從 c欄拉跳線到對應排數，再連通到 a欄的 ESP32 腳位。
> I2C 位址實測為 **0x27**（與 main.cpp 預設一致）。

## 蜂鳴器接線

蜂鳴器型號：**9042 被動式 3.3V**  
插在 **e欄 排27（正極）與 排30（負極）**

```
蜂鳴器腳位    插入位置     中繼           最終目的地
+（長針腳）   e欄排27      a欄排27        ESP32 懸空腳 排15（P13/GPIO13）
-（短針腳）   e欄排30                     右側電源軌 -
```

> **注意：** 被動式蜂鳴器需要 PWM 訊號驅動，程式使用 `ledcWrite()` 產生 2700Hz，不可用 `digitalWrite()`。

---

## 藍色按鈕接線（裝置開關）

按鈕跨插在溝槽兩側，插在 **排10 與 排12**。

```
接線位置      功能      最終目的地
c欄排10       訊號      ESP32 懸空腳 排13（P12/GPIO12）
c欄排12       GND       右側電源軌 -
```

> **程式設定：** `INPUT_PULLUP`，按下時為 LOW。  
> **功能：** 按一下開啟測量，再按一下關閉測量。

## 黃色按鈕（已移除）

> ❌ **黃色按鈕（查詢上次心律）已從專案移除。** 其接線反覆造成短路、干擾燒錄與正常開機（`flash read err` 重啟迴圈），且僅為附加功能，故決定移除。main.cpp 已刪除所有相關程式碼，硬體上黃色按鈕與其接線皆已拆除。

## 給 Claude Code 的腳位定義

```cpp
// AD8232
#define ECG_PIN   36   // SVP - OUTPUT
#define LO_PLUS   34   // SVN - LO+
#define LO_MINUS  35   // P34 - LO-

// LCD I2C
#define LCD_SCL   22   // P22
#define LCD_SDA   21   // P21

// 蜂鳴器（被動式，需PWM）
#define BUZZER    13   // P13
#define BUZZER_CH  0   // LEDC channel
#define BUZZER_HZ  2700 // 共振頻率

// 按鈕（黃色按鈕已移除）
#define BTN_BLUE   12  // P12 - 裝置開關
```
